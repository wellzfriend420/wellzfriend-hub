# ロードマップ

アプリ事業の成果目標と正式採用済みの取り組みを時間軸で整合させる正本です。[roadmap.yaml](roadmap.yaml)をポートフォリオ管理責任者が月次更新し、四半期承認します。

状態は `proposed`（検討中）、`committed`（正式採用）、`in_progress`（実行中）、`completed`（完了）、`cancelled`（中止）です。日程は約束と見込みを区別し、中止項目も理由を残します。改善は `IMP-NNNNNN`、対象は各台帳IDで参照します。

機密情報、個人名、顧客情報、非公開の契約・金額詳細は保存しません。サンプルは実在しません。

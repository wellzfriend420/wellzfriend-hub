# Governance

Hubの意思決定、責任、正本、変更管理を統治する領域です。会社ルールの詳細は[アプリ事業運営マニュアル](../docs/operations-manual/README.md)、管理対象の正本は[台帳](../registries/README.md)に置きます。

- 方針の承認責任は「アプリ事業責任者」、日常保守は「Hub管理責任者」とする。
- 重要な決定は目的、選択肢、決定、影響、承認役割、決定日を[意思決定記録](decision-log.yaml)へ残す。
- 検討案は確定文書へ混在させず `proposed` として管理する。
- GitHub上の実装手順は[GitHub運用ガイド](../docs/platform-guides/github/README.md)に分離する。

機密情報、個人情報、顧客固有情報は保存しません。

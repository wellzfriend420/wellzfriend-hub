# GitHub運用ガイド

アプリ事業運営マニュアルの会社ルールをGitHub上で実現するためのガイドです。GitHubが変わった場合は本ガイドを更新し、会社ルールは変更しません。

## 文書一覧

- [リポジトリとブランチ](repository-and-branches.md)
- [Pull RequestとCODEOWNERS](pull-requests-and-codeowners.md)
- [IssuesとADR](issues-and-adr.md)
- [Actionsとセキュリティ](actions-and-security.md)
- [リリースとタグ](releases-and-tags.md)

リポジトリの正本一覧は[リポジトリ台帳](../../../registries/repositories/README.md)です。秘密情報、顧客情報、接続情報をIssue、PR、ログ、成果物へ記録しません。

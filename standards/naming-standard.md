# 命名規則

> 状態: approved / 管理責任: 開発標準管理責任者 / 最終確認日: 2026-07-04

## 不変ID

| 対象 | 形式 |
|---|---|
| システム | `SYS-NNNN` |
| アプリ | `APP-NNNN` |
| データベース | `DB-NNNN` |
| 機能 | `FEAT-NNNNNN` |
| AI | `AI-NNN` |
| Business Template | `BT-NNNN` |
| 改善要望 | `IMP-NNNNNN` |

IDはゼロ埋め連番とし、一度割り当てたら名称変更・移管でも維持します。廃止IDは永久欠番です。表示名は利用者に理解できる語を使い、IDと併記します。

ファイル・ディレクトリ名は原則として小文字英数字とハイフンを使い、README等の慣例名は大文字を許可します。日付は `YYYY-MM-DD`、Versionは `X.Y.Z` とします。秘密値や個人名を名称へ含めません。

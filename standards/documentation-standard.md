# 文書標準

> 状態: approved / 管理責任: 文書管理責任者 / 最終確認日: 2026-07-04

- 文書は目的、対象、状態、管理責任役割、最終確認日、正本の所在を明確にする。
- 確定事項と未決事項を章または状態で分離する。
- 他の正本を複製せず、不変IDと相対リンクで参照する。
- 読者が判断・実行・検証できる具体性を持たせる。
- 変更理由と利用者影響を履歴へ残す。
- 廃止文書は後継先と理由を示して保管し、無断削除しない。
- 顧客情報、個人情報、接続情報、資格情報、秘密値を保存しない。

Markdownは見出しを一段ずつ使用し、リンク文言だけで行先が分かるようにします。例示は `【サンプル】` と明記し、架空の顧客情報を使いません。
